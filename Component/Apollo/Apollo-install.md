### 快速开始
`https://github.com/ctripcorp/apollo/wiki/Quick-Start` 根据文档。
* Java 1.8
* MySQL 5.6.5+
* 下载Quick Start安装包
* MySQL中创建需要的数据库： ApolloPortalDB， ApolloConfigDB
* Apollo配置数据库连接： 
* 启动Apollo，需要三个端口，默认8070,8080,8090
* 访问配置信息： `http://localhost:8070` ， 用户名apollo，密码admin
* 应用接入Apollo
